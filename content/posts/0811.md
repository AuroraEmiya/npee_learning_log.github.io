   |     算法     |              |  时间复杂度  |              |             |        |
   | :----------: | :----------: | :----------: | :----------: | :---------: | :----: |
   |              |   最好情况   |   平均情况   |   最坏情况   | 空间复杂度  | 稳定性 |
